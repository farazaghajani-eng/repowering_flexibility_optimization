# Dataset Description
## Overview
This folder contains input data used for power system repowering_flexibility analysis.

## Files

### 5_years_GBN_dataset.csv
- mid-Hourly load demand (MW)
 
This is for 5_years_demand_envelope.py

### ireland-2017_Data.csv
- Hourly load demand (MW)
 
### PUBLIC GEN DATA.csv
- different generation assets (MW)
 
Both for thesis_game_place.py
 
### bus-data_IEEE_118-bus.csv
- grid and 118-bus data
 
### demand-subperiod_IEEE_118-bus.csv
- mid-Hourly load demand (MW)

### net-load.csv
- mid-Hourly load demand (MW)
- wind uncertain generation data (MW)
 
### generation-tech-data_IEEE_118-bus.csv
- Capacity (MW)
- Ramp rates
- Operational constraints

### gen-hybrid_IEEE_118-bus.csv
- Generation Portfolio in IEEE 118 bus

### storage-generation_IEEE_118-bus.csv
- Capacity (MW)
- Ramp rates
- Operational constraints

### wind-generation_IEEE_118-bus.csv
- cost

all classified IEEE_118-bus data for comparision_rehab_vs_new.py and real_comparision_rehab_vs_new_.py

## Source
- Synthetic data based on typical power system scenarios
- Inspired by real-world operational characteristics

## Notes
All values are normalized and used for simulation purposes.
